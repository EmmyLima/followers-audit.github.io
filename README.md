# Instagram Follower Audit

A simple, privacy-focused tool to find out who isn't following you back on Instagram. This data never leaves your machine as it is not uploaded to a server.

## How to Use

### Download your Instagram Data
To use this tool, you need to request your data from Instagram:
1. Go to **Settings** > **Your Information and Permissions**.
2. Select **Download Your Information**.
3. Choose **Some of your information** and select **Followers and Following**.
4. **Important:** Set the format to **JSON** (not HTML) and select **From all time**.
5. Once the file arrives in your email, download and unzip it.
6. **following might show up as** following_1.json **so just rename it to** following.json
